Sobre mim

Sou estudante de Análise e Desenvolvimento de Sistemas e trabalho como Mecânico de Autos.
Atualmente, trabalho na manutenção de veículos automotores de linha leve, realizando diagnósticos, programação de módulos de transmissão, reparos e orçamentos, além disso, trabalho com projetos pessoais sobre Ciência de Dados, para adquirir experiência na solução de problemas de negócio e domínio sobre as ferramentas de análise de dados. Estou buscando uma oportunidade de trabalhar profissionalmente como Cientista de Dados para melhorar a tomada de decisão da empresa, através da construção de soluções usando dados.

Experiências

Construção de soluções de dados para problemas de negócio, próximos dos desafios reais das empresas, utilizando dados públicos de competições de Ciência de Dados, onde eu abordei o problema desde a concepção do desafio de negócio até a publicação do algoritmo, utilizando ferramentas de Cloud Computing.

12+ Como Mecânico de Autos

Frequentemente para realizar as manutenções nos veículos, primeiramente estudo o histórico de reparos, possíveis causas, o que já foi substituído, averbalizado propriamente pelo cliente ou em ordem serviços ja finalizadas. Assim, buscando uma maior eficácia no reparo e solucionando o problema do cliente.
